# Interactive-Sales-Analytics-Dashboard-in-Excel
#  Sometimes the simplest tools (Excel) can deliver the most powerful insights!
## Overview
An interactive Excel dashboard that provides comprehensive insights into sales performance, customer behavior, delivery metrics, and product demand patterns.

## 🎯 Project Objectives
- Visualize sales trends over 13 weeks
- Analyze customer demographics and purchasing patterns
- Track delivery performance across regions
- Monitor customer satisfaction ratings
- Identify top-demand products
- Compare performance across different order channels

## 📈 Key Features

### Dashboard Components:
1. **KPI Summary Panel**
   - Total Orders: 2,400
   - Total Quantity: 11,997 units
   - Total Revenue: $649.0K
   - Average Rating: 4.0/5
   - Average Delivery Time: 2.3 days

2. **Trend Analysis**
   - 13-week quantity vs amount comparison
   - Week-over-week performance tracking

3. **Customer Insights**
   - Gender distribution across platforms (App, Instagram, Partner App, Target.com, Website)
   - Purchase quantity patterns
   - Customer satisfaction scores

4. **Product Analytics**
   - Most demanded products (T-Shirts, Sneakers, Bikinis, etc.)
   - Product category performance

5. **Geographic Analysis**
   - Delivery location heat map
   - Regional performance tracking (Andhra Pradesh focus)

6. **Operational Metrics**
   - Shopping time analysis
   - Delivery efficiency tracking

## ️ Tools & Technologies
- **Microsoft Excel** - Dashboard creation
- **Pivot Tables** - Data aggregation
- **Charts & Visualizations** - Bar charts, line graphs, maps, donut charts
- **Slicers** - Interactive filtering by Order Mode and Gender

## 📊 Data Structure
The dataset includes:
- Transaction ID
- Product details
- Quantity & Pricing
- Order & Ship Dates
- Customer Gender
- Order Mode (App, Instagram, Partner App, Target.com, Website)
- Customer Ratings (1-5)
- Geographic Data (State, County)
- Delivery Metrics
- Week Numbers

##  Key Insights

### Platform Performance:
- **App** leads with 19.0% female and 12.3% male customers
- **Website** shows strong engagement (12.5% female, 9.1% male)
- **Instagram** has lower conversion (5.0% female, 4.3% male)

### Customer Behavior:
- Most customers buy 1-2 items per order
- Group orders show significant volume
- Peak satisfaction rating: 4.0/5

### Top Products:
- T-Shirts
- Sneakers
- Bikinis
- Sundresses
- Hoodies & Sweatshirts

## 🚀 How to Use
1. Open `Sales_Dashboard.xlsx` in Microsoft Excel (2016 or later)
2. Use slicers on the left panel to filter by:
   - Order Mode (App, Instagram, Partner App, Target.com, Website)
   - Gender (Female, Male, Other, Unknown)
3. Hover over charts for detailed tooltips
4. Explore different tabs for specific insights
